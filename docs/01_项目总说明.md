# 01 · 项目总说明

## LoveHouse 是什么

LoveHouse 是一个个人数字空间（情侣向），属于小婷和小克的小屋。
用来记日记、存记忆、贴语录、管待办、记心情、发动态。

## 核心设计原则

**每个部分都是独立零件，可以单独替换，互不耦合。**

- 换数据库 → 只改 `src/core/supabase.js`
- 加功能 → 在 `src/modules/` 新增一个目录
- 加主题 → 在 `src/themes/` 新增一个 CSS 文件
- 改导航/外壳 → 只动 `src/shared/AppShell.jsx`

## 技术栈

| 部分 | 选型 |
|------|------|
| 前端框架 | React + Vite |
| 路由 | react-router-dom（HashRouter，兼容 GitHub Pages） |
| 数据库 | Supabase（项目ID: cvyguanuaxcypsvoozeo，新加坡区） |
| 部署 | GitHub Actions 自动构建 → GitHub Pages |
